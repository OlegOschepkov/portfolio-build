Примеры работ

  • Учебный проект "Девайс", HTML + CSS

    Самый сложный проект среди предлагаемых на курсе "первый уровнь HTML&CSS". Сложная сетка, неочевидный порядок компонентов в разметке, применются градиенты и трансформации. В проекте две страницы - главная и каталог. Слайдеры на главной реализованы на чистом CSS.

  • Учебный проект "Мишка"*, HTML + Less + Gulp. Позже заменил Gulp на Grunt

    Самый сложный проект среди предлагаемых на на курсе "второй уровнь HTML&CSS". Множество декоративных элементов, нестандартная сетка, сложные перестроения всех блоков, изменения размеров фотографий. 3 размера - мобильный, планшетный, десктоп. 3 страницы - главная, страница с товарами и формой. Применяется адаптивная и ретиновая графика, спрайт.

    * Переход на собранную страницу. Оригинал кода тут

  • Учебный проект "Кекстаграмм", JS

    Проект курса "JavaScript, 1 уровень". Проект позволяет загрузить свою фотографию, применить фильтр, изменить его насыщенность, размер фото, в полях хештега и кмментария осуществляется проверка на валидацию. Также можно посмотреть фотографии других пользователей (комментарии и количество лайков храняться на сервере), есть функция сортировки. Для отрисовки фотографий применяется шаблон.

  • Учебный проект "Код-и-магия", JS

    Проект курса "JavaScript, 1 уровень". В этой игре вы можете передвигать персонажа, стрелять фаерболом, изменить имя, цвет глаз, плаща, фаербола. Также выводиться динамическая таблица результатов, и можно сравнить своего персонажа с другими (персонажи подгружаются с сервера).

  • "Девайс" на Wordpress

    Перенес Девайс на Wordpress. Просто было любопытно посмотреть как работает CMS. Из панели управления можно изменить слайды (реализован только в отношении первого слайда), добавить\изменить товары в каталоге. Добавил несколько PHP скриптов (генерация и автоподбор ссылок, созадние списка товаров, получение занчений полей, отправка сообщений и прочее), хидер и футер вынес в отдельные файлы.
